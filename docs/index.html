CORTEX on GitHub Pages

This page is the static GitHub Pages frontend for CORTEX. GitHub Pages cannot run Python, so the actual CORTEX engine must be hosted on a public backend. Use the API URL below to connect the frontend to your deployed CORTEX backend.

Backend endpoint

This page is preconfigured to use the public CORTEX backend at https://cortexwebbackend.onrender.com.

Visitors do not need to enter any backend URL. The frontend will always use the built-in public backend.

Ask CORTEX

Response will appear here.

Use Ctrl+Enter to submit from the textarea.

GitHub Pages setup

To make this available on the internet via GitHub Pages:

  1. Copy this `docs/index.html` to your repository.
  2. Enable GitHub Pages for the repo, using the docs/ folder as the source.
  3. Deploy `web/server.py` to a public host such as Render, Railway, or a VPS.
  4. Set the backend URL above to the public server address.

Once deployed, the page can be opened from the GitHub Pages URL and used from anywhere.
