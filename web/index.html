How to use

To use CORTEX from GitHub Pages without downloading it:

  1. Host this repo's `web/` folder on GitHub Pages.
  2. Deploy the Python backend using `web/server.py` to a public server or cloud VM.
  3. Set the backend URL above to your deployed server.

GitHub Pages provides only the static frontend. The actual CORTEX engine must run on a separate backend service.
